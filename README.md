# User Tests

UI + API тесты для управления пользователями.

## Требования

- Java 11 или выше

## Запуск тестов

### Через IDE (IntelliJ IDEA)

1. Откройте проект в IDEA
2. Перейдите в класс `UserTests`
3. Нажмите правой кнопкой мыши → `Run 'UserTests'`

### Через командную строку

```bash
./gradlew test
